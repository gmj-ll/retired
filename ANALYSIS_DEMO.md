# 分析页面功能演示

## 功能概览

分析页面现已实现双圆环设计，为用户提供直观的时间分配可视化体验。

## 主要功能

### 1. 当日时间分析圆环
- **功能**: 显示今日工作、睡眠、自由时间分配
- **特点**: 
  - 三色分段圆环显示
  - 中心数据详情展示
  - 百分比和小时数双重显示
  - 响应式设计适配不同屏幕

### 2. 职业生涯分析圆环
- **功能**: 显示整个职业生涯的时间分配预测
- **特点**:
  - 基于用户退休政策的精确计算
  - 动态进度箭头指示当前阶段
  - 多时间单位切换（年/月/日/时）
  - 智能时间洞察提示

### 3. 交互体验
- **手势导航**: 左右滑动切换分析视图
- **时间单位**: 动态切换显示格式
- **进度指示**: 底部圆点显示当前页面
- **平滑动画**: 页面切换和数据更新动画

## 技术亮点

### SVG 圆环渲染
- 使用 react-native-svg 实现高质量圆环图表
- 精确的角度计算和路径描述
- 支持自定义颜色主题

### 智能数据计算
- 基于真实退休政策的时间预测
- 考虑工作年限和延迟退休政策
- 动态进度计算和显示

### 响应式设计
- 适配不同屏幕尺寸
- 灵活的布局系统
- 优化的触摸交互

## 使用指南

### 查看当日分析
1. 打开分析页面，默认显示当日圆环
2. 查看工作、睡眠、自由时间的分配情况
3. 点击"查看历史"按钮可访问历史数据（待实现）

### 查看职业生涯分析
1. 向右滑动或点击进入职业生涯分析
2. 使用时间单位选择器切换显示格式
3. 观察进度箭头了解当前职业阶段
4. 阅读智能洞察获取时间使用建议

### 个性化设置
- 颜色主题：支持自定义三种时间类型的颜色
- 显示偏好：可选择百分比或具体时间显示
- 时间精度：调整显示的小数位数
