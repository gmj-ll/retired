# 首页卡片优化

## 修复的问题

### 🐛 **ImagePicker API 错误修复**
- **问题**: `ImagePicker.MediaType.Images` API 已过时，导致选择背景图片时报错
- **修复**: 更新为 `ImagePicker.MediaTypeOptions.Images`
- **影响**: 现在可以正常选择和设置背景图片

### 📝 **文案顺序优化**
- **问题**: 原来的文案顺序不够直观
- **修复**: 将"距离退休还有"移到最上方，形成更自然的阅读顺序

#### 优化前的顺序：
```
[大数字]
[单位]
距离退休还有
[提示文字]
```

#### 优化后的顺序：
```
距离退休还有
[大数字]
[单位]
[提示文字]
```

### 🎨 **默认背景样式优化**

#### 视觉改进：
1. **渐变色彩更新**：
   - 从 `['#667eea', '#764ba2']` 改为 `['#4facfe', '#00f2fe']`
   - 新配色更加清新明亮，符合现代设计趋势

2. **卡片尺寸增加**：
   - 最小高度从 200px 增加到 220px
   - 提供更好的视觉比例

3. **遮罩层优化**：
   - 背景透明度从 0.5 增加到 0.6，提高文字可读性
   - 内边距从 30px 增加到 35px
   - 圆角从 15px 增加到 18px
   - 最小宽度从 80% 增加到 85%

#### 文字样式增强：
4. **标题文字**（"距离退休还有"）：
   - 字体大小从 16px 增加到 18px
   - 添加字母间距 0.5px
   - 颜色调整为 `rgba(255, 255, 255, 0.9)`
   - 底部间距增加到 15px

5. **倒计时数字**：
   - 字体大小从 48px 增加到 52px
   - 添加文字阴影效果
   - 增强视觉冲击力

6. **单位文字**：
   - 字体大小从 20px 增加到 22px
   - 字重增加到 600
   - 添加文字阴影
   - 上下间距优化

7. **提示文字**：
   - 字体大小从 11px 增加到 12px
   - 透明度从 0.7 增加到 0.8
   - 添加字母间距 0.3px
   - 底部间距增加到 20px

#### 阴影和层次：
8. **卡片阴影**：
   - 为遮罩层添加阴影效果
   - 增强卡片的层次感和立体感

## 技术细节

### API 更新
```typescript
// 修复前（报错）
mediaTypes: ImagePicker.MediaType.Images

// 修复后（正常）
mediaTypes: ImagePicker.MediaTypeOptions.Images
```

### 渐变配色
```typescript
// 新的渐变配色
colors={['#4facfe', '#00f2fe']}
```

### 样式优化
```typescript
countdownOverlay: {
  backgroundColor: 'rgba(0, 0, 0, 0.6)', // 增强对比度
  padding: 35,                           // 增加内边距
  borderRadius: 18,                      // 更大圆角
  // ... 添加阴影效果
}
```

这些优化确保了首页卡片在功能性和美观性方面都有显著提升，为用户提供更好的视觉体验和交互体验。
